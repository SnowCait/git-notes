## コマンドライン
- 基本
- 何でもできる
- `git graph` エイリアスは設定しておく

## GitHub Desktop
- シンプル
- [github.com](https://github.com/) からクローンができる

## Fork
- 一通りの機能が揃っている
- rebase 時にバックアップブランチを作成してくれる
- ググラビリティが低い
- `git remote` ができない
- 内臓 Git を使うと重いので OS 側の Git を指定する

## SourceTree
- 一通りの機能が揃っている
- 重い
