## Fork
- 一通りの機能が揃っている
- rebase 時にバックアップブランチを作成してくれる
- ググラビリティが低い
- `git remote` ができない
- 内臓 Git を使うと重いので OS 側の Git を指定する
- [Git クライアント Fork がとてもいい – Numb.](https://wp.graphact.com/2020/04/29/git-client-fork/)
